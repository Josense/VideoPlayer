# VideoPlayer

A Simple Online mp4 player based Exoplayer library

* 给定正确的MP4网址，可以自动跳转播放；
* 代码内部自定义了快进/后退15秒；
* 首页可自定义缓冲区大小（最小缓冲区设置为了50秒，这里自定义的是最大缓冲区大小）；
